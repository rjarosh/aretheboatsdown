What is this?

I use python, real-time lightning data, image processing, and some possibly misguided assumptions about Disney's strategy on weather safety to guess whether the boats at WDW are running.


The backend script checks for lightning every 60 seconds within a ~7 mile radius from Epcot (as an arbitrary center point). Status changes are pushed out using the Twitter API. A lightning strike detected automatically starts a 30-minute cooldown timer before the script runs again. This is clearly for entertainment/hobby purposes only and you shouldn't actually use this to know if the weather is bad or to make the decision to walk a mile back to your hotel instead of waiting for a boat.
